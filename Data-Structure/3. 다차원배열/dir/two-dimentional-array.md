# 2차원 배열
## 개념
> 행과 열로 이루어진 배열
## 선언
- 자료형 이름[배열길이][배열길이];
- int array[4][3];
## 배열의 논리적 구조와 물리적 구조
- 크게 다르지 않음
- 행 우선 순서
- 열 우선 순서
## 배열의 초기화
- int array[4][3] = {1, 2, 3}, {1, 2, 3}, {9, 8}, {10};
- ex)

|1|2|3|
